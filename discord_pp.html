Vortex Bot Privacy Policy

Your privacy and data protection are important to us

Effective Date: July 17, 2025

This document explains how Vortex Bot handles data when used in Discord servers. We are committed to protecting your privacy and being transparent about our data practices.

1 What We Collect

Vortex Bot collects and uses the following data during its operation:

User Information: Discord user ID, username, and discriminator.
Guild & Channel Data: Server ID, name, and the channel used to invoke a command.
Command Usage Logs: Timestamps, command names, and arguments (e.g., city for weather lookups).
Weather Lookup Data: API queries to OpenWeatherMap (city names only, not user information).
Temporary Caching: Cached weather data for performance (stored for up to 30 minutes).
Console Commands (admin only): Server-wide broadcasting or command viewing via the console interface.
No personal information beyond what is provided by Discord is collected or permanently stored.

2 How Data Is Used

The data we collect is used for the following purposes:

  • To operate features like weather, timezones, and user/server info.
  • To log and debug bot behavior (e.g., error handling or command failures).
  • To prevent command spamming via cooldowns.

3 Data Retention

We follow strict data retention policies to protect your information:

Logs: Most data is not stored persistently, except logs which are rotated and retained for up to 30 days.

Cache: Cached weather responses are cleared automatically after a set time.

4 Data Sharing

We take data sharing seriously and follow these principles:

  • No user data is ever sold or shared with third parties.
  • Data may be shared only to comply with legal obligations or Discord's requests.

5 Your Rights

You have the following rights regarding your data:

  • Data Deletion: Request deletion of any logged data associated with your Discord user ID.
  • Data Inquiry: Request clarification of what information has been collected about you.
